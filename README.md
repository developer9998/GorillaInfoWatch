# GorillaInfoWatch

*GorillaInfoWatch* is a general purpose utility mod for *Gorilla Tag*.

## Features

*GorillaInfoWatch* contains features
*GorillaInfoWatch* contains many features that makes it different from any other watch mod you can use.

### Scoreboard

The Scoreboard screen featured in *GorillaInfoWatch* is stylized with unique art assets and functionality to inspect:

- **Room**<br>
  Includes room name and player count, region and game mode, with other related indicators

- **Player**<br>
  Includes player name, avatar colour, account creation date, mute and friend options, a report selection dropdown, with a significance viewer when compatible
  
### Details

The Details screen featured in *GorillaInfoWatch* consists of a single page with the following indicators:

- **Appearance**<br>
  Includes player name, avatar colour, and account creation date

- **Economy**<br>
  Includes shiny rock balance and total number of purchased cosmetics

- **Statistics**<br>
  Includes total number of players tagged, tags recieved, and rounds played, separated by game mode, excluding casual-based modes

### Friends

The Friends screen featured in *GorillaInfoWatch* is based on the friend machine provided by the game and therefore acts about the same.

However, with the Friends screen, player names are guaranteed to show up, unlike the friend machine after a friend remains offline for over a day.

### Mod Status

The Mod Status screen featured in *GorillaInfoWatch* will list your installed mods with a couple of additions:

- **Toggle Support**<br>
  When supported by a mod, a switch will be placed beside its entry that can toggle the mod

- **Configuration**<br>
  When supported by a mod, the ability to read all configuration and write configuration with certain types is provided

The means of how your installed mods are ordered depends on various indicators, including toggle support, configuration entry count, and alphabetical order based on the name of the mod.

### Shortcuts

The shortcut feature in *GorillaInfoWatch* provides access to quick methods that can be ran through the menu or bound to the watch.

This feature is still in development, though the current list of shortcuts are as follows:

- **Join Random Room**<br>
  Joins a random public room under a map that is loaded

- **Join Specific Room**<br>
  Joins a specific room of the name provided by the player

- **Leave**<br>
  The current room containing the player is left

- **Rejoin**<br>
  The current room containing the player is re-joined (left and then joined)

- **Copy Room Name**<br>
  Copies the name of the current room to your clipboard

### Settings

The Settings screen in *GorillaInfoWatch* is separated by category and includes configuration native to the mod.

This feature is still in development, though the current list of settings are as follows:

- **Preferred Hand / Orientation**<br>
  Which hand the watch is placed on

- **Notification Sources**<br>
  The list of sources you have allowed to send their respective notifications

- **Notification Haptic Amplitude**<br>
  The amplitude of the haptic sent when a notification is recieved

- **Notification Haptic Duration**<br>
  The duration of the haptic sent when a notification is recieved

- **Silent Notification Haptic Amplitude**<br>
  The amplitude of the haptic sent when a silent notification is recieved

- **Silent Notification Haptic Duration**<br>
  The duration of the haptic sent when a silent notification is recieved

- **Notification Sound Volume**<br>
  The volume of the sound produced when a notification is recieved

- **Shortcut Hold Duration**<br>
  The maximum duration needed to activate a shortcut using the shortcut button

- **Shortcut Interval**<br>
  The minimum interval between shortcut activation using the shortcut button

- **Show Public Name**<br>
  Whether room names under public visibility are shown

- **Show Private Name**<br>
  Whether room names under private visibility are shown

### Credits

The Credits screen in *GorillaInfoWatch* admires the founders, contributors, and testers of the mod, with supporters detailed as well:

- Five members each from the "Basic" and "Dweller" tiers are selected daily and shown
- All members from the "Prestige" tier and shown on the screen

### Player Significance
Each player in the game, including yourself, will likely have some form of significance, ranging anywhere from room master, to representing a group, and even as a recognized user.

The mod uses GorillaFriends to expand significance to including friends, verified (in light green), and recently played with users (in light red).

### Notifications
It's always nice to be notified of information when you need it the most, and GorillaInfoWatch has you covered!

Examples of notifications the player could recieve include when certain players (with significance) join or leave the current room, room join failures, server messages, and quest completions.

### Custom Screens
The majority of the features listed prior to this are known as "screens", though built-in to the mod, don't differ much from custom screens.

With knowledge in mod development and the infrastructure of GorillaInfoWatch, developers can create their own custom screens using widgets and utilites to be run by GorillaInfoWatch.

## Usage
To use GorillaInfoWatch, press on the face of the info watch located on your left hand, and interact with the menu using various widgets and set buttons at the bottom.
